Questa libreria fornisce l\textquotesingle{}implementazione di una lista concatenata singola, con la possibilità di inserire, eliminare, cercare nodi e rimuovere duplicati. La lista è composta da nodi che memorizzano un valore intero e un puntatore al nodo successivo.
