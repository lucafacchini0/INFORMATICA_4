Detailed Description

Implementazione di una lista concatenata.

La classe Lista fornisce operazioni per manipolare una lista concatenata singola. È possibile inserire nodi in testa o in coda, eliminare nodi, cercare un valore, visualizzare la lista, contare il numero di nodi e rimuovere i duplicati. La lista può contenere valori interi.

Distruttore della classe Lista.

Il distruttore dealloca tutta la memoria utilizzata dai nodi della lista concatenata, liberando le risorse. Viene invocato automaticamente quando l'oggetto Lista esce dallo scope.
