Detailed Description

Definizione della struttura dati Lista Concatenata e della classe Nodo.

Author
Luca Facchini

Questa libreria fornisce l'implementazione di una lista concatenata singola, con la possibilità di inserire, eliminare, cercare nodi e rimuovere duplicati. La lista è composta da nodi che memorizzano un valore intero e un puntatore al nodo successivo.
